# Arduino UNO R3

* pin 19 => SCL (MISO)
* pin 18 => SDA (MOSI)
* 5V     => 5V
* GND    => GND
* pin 08 => KY-040 SW
* pin 07 => KY-040 CLK
* pin 06 => KY-040 DT


# Arduino Nano

* pin 04 => GND
* pin 27 => 5V
* pin 24 => A5 / SCL
* pin 23 => A4 / SDA
* pin 09 => D6 / KY-040 DT
* pin 10 => D7 / KY-040 CLK
* pin 11 => D8 / KY-040 SW
